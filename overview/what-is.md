# 什么是流体力学

流体力学是研究**流体（液体和气体）的宏观运动规律及其与固体相互作用**的科学。它是经典力学的重要分支，也是现代工程技术的理论基础之一。

## 流体的定义

流体与固体的本质区别在于受力时的响应：
- **固体**：在剪应力下产生有限变形
- **流体**：在任何微小的剪应力下都会持续变形（流动）

## 核心研究问题

1. **流体的运动规律** — 速度场、压力场、温度场的分布与演化
2. **流体与固体之间的作用力** — 升力、阻力、力矩
3. **流体内部的相互作用** — 湍流、涡旋、激波
4. **能量传递** — 对流换热、混合
5. **多物理场耦合** — 流体-结构、流体-电磁、化学反应的耦合

## 应用领域

| 领域 | 典型问题 |
|------|----------|
| 航空航天 | 机翼设计、高超声速再入 |
| 船舶海洋 | 船体阻力、波浪载荷 |
| 能源动力 | 风力机、水力发电、核反应堆冷却 |
| 生物医学 | 血液流动、呼吸力学 |
| 环境工程 | 污染物扩散、大气环流 |
| 材料加工 | 注塑成型、焊接 |
| 土木工程 | 风荷载、水坝应力 |

## 研究方法

- **理论分析**：从NS方程出发，简化求解
- **实验研究**：风洞、水槽、PIV测速
- **数值模拟（CFD）**：用计算机离散求解控制方程
