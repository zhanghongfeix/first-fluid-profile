# 分支领域

## 按流动特性分类

| 分支 | 研究对象 |
|------|----------|
| 不可压缩流 | 低速流动，$\nabla \cdot \mathbf{u} = 0$ |
| 可压缩流 | 高速流动，密度变化不可忽略 |
| 层流 | 规则、可预测的流动 |
| 湍流 | 混沌、多尺度、统计描述的流动 |
| 牛顿流体 | 应力与应变率成正比 |
| 非牛顿流体 | 应力-应变率关系非线性 |

## 按应用场景分类

| 分支 | 应用 |
|------|------|
| 空气动力学 | 飞行器、风力机 |
| 水动力学 | 船舶、海洋平台 |
| 多相流 | 化工、核能 |
| 生物流体力学 | 血液循环、呼吸 |
| 环境流体力学 | 大气、海洋、污染物 |
| 微流体 | 芯片实验室、微纳器件 |
| 磁流体力学 | 等离子体、地核动力学 |

## 按研究方法分类

| 分支 | 方法 |
|------|------|
| 理论流体力学 | 数学分析、摄动法 |
| 实验流体力学 | 风洞、水槽、PIV/LIF测量 |
| 计算流体力学 (CFD) | 有限体积法、有限元法、谱方法 |
