# 反应流与燃烧

伴随化学反应的流体流动，涉及流体力学、热力学和化学动力学的强耦合。

## 概述

化学反应的特征时间尺度（$\tau_c$）与流动时间尺度（$\tau_f$）的对比：

| Damköhler 数 | 含义 |
|---------------|------|
| $Da = \tau_f / \tau_c \gg 1$ | 反应快，由混合控制 |
| $Da = \tau_f / \tau_c \ll 1$ | 反应慢，由化学动力学控制 |

## 研究现状

### 燃烧模式

| 模式 | 特点 |
|------|------|
| 预混燃烧 | 燃料+氧化剂预混合，火焰管式传播 |
| 非预混燃烧（扩散火焰） | 边混合边燃烧 |
| 部分预混 | 两者之间 |

### 数值模拟方法

| 方法 | 特点 |
|------|------|
| 火焰面模型 (Flamelet) | 假设火焰是薄的反应-扩散层 |
| 条件矩封闭 (CMC) | 条件平均推导 |
| 输运 PDF 方法 | 随机描述湍流-化学反应耦合 |
| 详细化学反应加速 | 自适应制表、ANN 替代、PCA 降维 |

### 爆震

- 爆震波结构：前导激波 + 反应区
- 旋转爆震发动机 (RDE)：连续旋转爆震波
- 爆震→爆燃转捩 (DDT)

## 关键技术挑战

1. 详细化学机理的降维（几百个物种→少数反应进度变量）
2. 湍流-化学非线性耦合
3. 碳烟 (Soot) 形成与辐射
4. 超临界燃烧

## 发展趋势

- 氢/氨燃烧（无碳燃料）
- AI 辅助化学机理优化
- 超声速燃烧（超燃冲压发动机）
- 电推进/混合推进与燃烧系统的一体化仿真
