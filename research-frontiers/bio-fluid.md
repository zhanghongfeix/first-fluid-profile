# 生物流体力学

将流体力学原理应用于生物学和医学问题的跨界学科。

## 概述

主要研究体系：

| 系统 | 典型问题 |
|------|----------|
| 心血管 | 动脉血流、动脉瘤、血栓 |
| 呼吸 | 气道气流、颗粒沉积 |
| 细胞尺度 | 红细胞变形、白细胞黏附 |
| 运动 | 鱼类游动、鸟类/昆虫飞行 |
| 细胞运动 | 鞭毛/纤毛驱动的微流动 |
| 生殖 | 精子运动、胚胎培养液流动 |

## 研究现状

### 心血管流体力学

- **Womersley 数** $Wo = R\sqrt{\omega/\nu}$ 表征脉动性
- 壁面剪应力 (WSS) 与动脉粥样硬化的关联
- FSI（流固耦合）模拟血管变形
- 4D-Flow MRI 提供活体内流场测量

### 游泳与飞行

- 波动游泳（鳗鱼式）vs 振荡游泳（金枪鱼式）
- 尾涡结构：反卡门涡街产生推力
- 扑翼飞行的前缘涡增升机制

## 发展趋势

- 个性化血管模型用于手术规划
- 仿生水下/飞行器设计
- 微纳游动机器人（药物输送）
- 组织工程中的灌注生物反应器设计
