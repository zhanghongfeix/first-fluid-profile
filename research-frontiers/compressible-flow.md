# 可压缩流动

密度变化不可忽略的流动，涉及激波、膨胀波等独特现象。

## 概述

可压缩性判断标准 — 马赫数：

$$Ma = \frac{U}{c}, \quad c = \sqrt{\gamma R T}$$

$Ma > 0.3$ 时，密度变化通常不可忽略。

## 研究现状

### 激波动力学

- 正激波 / 斜激波关系（Rankine-Hugoniot 条件）
- 激波-边界层相互作用（SBLI）：引起分离和低频振荡
- 激波-激波相互作用：Edney 分类 VI 型

### 激波管与激波风洞

- 强激波驱动的高焓流动
- 激波管中的 Richtmyer-Meshkov 不稳定性

### 高超声速流动

- 真实气体效应（振动激发、离解、电离）
- 气动加热预测
- 烧蚀热防护

## 关键技术挑战

1. SBLI 低频非定常性的物理机制和预测
2. 激波捕捉格式的数值耗散与色散权衡
3. 高温气体的热化学非平衡建模
4. 激波噪声/声爆的预测与抑制

## 发展趋势

- 第六代战斗机的高机动气动设计
- 高超声速民用运输（点对点亚轨道飞行）
- 火星进入/下降/着陆 (EDL) 的低密度气动
- 激波-湍流 DNS 数据库建设
