# 水动力学

研究水（等液体）的运动规律及与结构相互作用的学科。

## 概述

水动力学与空气动力学的本质区别：

| 特性 | 空气 | 水 |
|------|------|-----|
| 密度比 | 1 | ~1000 |
| 自由面 | 无 | 有 |
| 空化 | 无 | 有 |
| 波浪载荷 | — | 重要 |

## 研究现状

### 船舶水动力学

- 兴波阻力理论（开尔文船波）
- 球鼻艏设计减小船行波
- 螺旋桨空化与水弹性

### 海洋工程水动力学

- 波浪-结构相互作用
- 涡激振动 (VIV) 对隔水管和系泊缆的影响
- 浮式风机平台动态响应
- 海啸/风暴潮的近岸演化

### 自由面流动

- 破碎波动力学
- 液舱晃荡 (sloshing)
- 入水冲击 (water entry)

## 关键技术挑战

1. 空化的多尺度模拟（气泡成核→生长→溃灭）
2. 两相湍流界面追踪（VOF/Level Set 方法）
3. 非线性波浪与结构的强耦合
4. 尺度效应：模型试验→实船换算

## 发展趋势

- 数字孪生船舶（实时 CFD + 传感器数据）
- 仿生水下航行器（鱼/鲸/蝠鲼推进）
- 波浪能装置水动力优化
