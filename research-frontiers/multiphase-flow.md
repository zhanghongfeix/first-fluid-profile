# 多相流

同时存在两种或以上不同相态（气、液、固）的流动。

## 概述

常见多相流体系：

| 类型 | 例子 |
|------|------|
| 气-液 | 沸腾、空化、气泡柱 |
| 液-液 | 油水分离、乳浊液 |
| 气-固 | 流化床、气力输送 |
| 液-固 | 泥沙输运、水力输送 |
| 气-液-固 | 三相流化床 |

## 研究现状

### 数值方法

| 方法 | 特点 |
|------|------|
| 欧拉-拉格朗日 | 连续相 Euler，离散相追踪粒子 |
| 欧拉-欧拉（双流体模型） | 两相均视为连续介质 |
| VOF（Volume of Fluid） | 界面捕获，体积分数对流 |
| Level Set | 隐式界面追踪 |
| 相场法 | 扩散界面模型 |

### 关键现象

- **Rayleigh-Taylor 不稳定性**：重流体加速进入轻流体
- **Kelvin-Helmholtz 不稳定性**：两相界面速度差导致
- **Plateau-Rayleigh 不稳定性**：液柱断裂成液滴
- **Marangoni 效应**：表面张力梯度驱动流动

## 关键技术挑战

1. 跨尺度耦合：气泡/液滴从微米到米跨越 6 个数量级
2. 界面力模型（曳力、升力、虚拟质量力）
3. 相变过程（蒸发/凝结/凝固）的准确建模
4. 颗粒-湍流相互作用

## 发展趋势

- 高保真界面解析模拟（DNS of multiphase）
- 机器学习替代两相闭合模型
- 微重力多相流（空间推进剂管理）
- 多相流拓扑优化
