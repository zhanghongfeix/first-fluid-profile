# 连续介质假设

## 基本概念

连续介质假设是流体力学最基本的假设：**将流体视为连续分布的物质，忽略其分子离散结构**。

## 适用条件

用**克努森数**来判断：

$$Kn = \frac{\lambda}{L}$$

- $Kn < 0.001$：连续介质，NS方程适用
- $0.001 < Kn < 0.1$：滑移流，NS方程加滑移边界
- $0.1 < Kn < 10$：过渡区，需用Burnett方程或DSMC
- $Kn > 10$：自由分子流，玻尔兹曼方程

## 控制体与物质导数

连续介质力学中，用**控制体**（空间固定的区域）或**物质体**（跟随流体运动的团块）来分析问题。

流体微团的加速度由**物质导数**给出：

$$\frac{D\mathbf{u}}{Dt} = \frac{\partial \mathbf{u}}{\partial t} + (\mathbf{u} \cdot \nabla)\mathbf{u}$$
