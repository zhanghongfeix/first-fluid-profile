# ANSYS Fluent 基础

工业界使用最广泛的商业 CFD 软件。

## 核心流程

```
几何建模         网格生成         前处理           求解             后处理
   │               │               │               │                │
SpaceClaim  →  Fluent Meshing → Fluent Setup → Fluent Solver → CFD-Post
  /DesignModeler                 设置物理模型    求解器计算      可视化结果
```

## 第一步：几何与网格

- 导入 CAD 几何体（.step, .igs）
- 创建流场区域（Fluid Domain）
- 命名边界（inlet, outlet, wall, symmetry 等）
- 网格类型选择：

| 网格类型 | 优点 | 缺点 |
|----------|------|------|
| 四面体 | 自动化程度高 | 精度较低 |
| 六面体 | 精度高，数量少 | 手工工作量大 |
| 多面体 (Poly-Hexcore) | 兼顾两者 | Fluent Meshing 特有 |

## 第二步：物理模型设置

**推荐路径**：

1. 启用能量方程（如有换热）
2. 选择湍流模型：
   - $k\text{-}\varepsilon$ Realizable：通用（分离流谨慎）
   - $k\text{-}\omega$ SST：推荐首选（壁面到远场统一处理）
3. 定义流体材料（密度、粘度等）
4. 设置边界条件

## 第三步：求解

- **初始化**：Standard / Hybrid / FMG
- **监视器**：残差 + 关键物理量（升力系数 $C_l$、阻力系数 $C_d$）
- **收敛判断**：
  - 残差下降 3-4 个数量级
  - 监视量稳定（不随时间/迭代变化）
  - 通量平衡（质量/能量通量守恒）

## 第四步：后处理

CFD-Post 或 Fluent 内置后处理：
- 截面云图（压力、速度、温度）
- 流线（Streamline）
- 等值面（Iso-surface λ₂ 准则显示涡结构）
- 报告升/阻力系数

## 常见问题

| 问题 | 解决方向 |
|------|----------|
| 残差不降 | 减小松弛因子、改善网格质量 |
| 发散 | 检查边界条件是否有误、降低库朗数 |
| 结果物理不合理 | 检查网格无关性、换湍流模型 |
