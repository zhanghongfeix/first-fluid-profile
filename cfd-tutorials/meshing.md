# 网格生成基础

网格是 CFD 精度的基石——"好网格 + 低阶格式 > 差网格 + 高阶格式"。

## 网格类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 结构化网格 | 规则排列，计算快 | 简单几何 |
| 非结构化网格 | 灵活填充任意区域 | 复杂几何 |
| 混合网格 | 边界层结构化 + 远场非结构 | 大多数工程问题 |
| 重叠网格 (Overset) | 多套网格插值 | 运动物体 |
| 笛卡尔网格 | 直角切割 + 浸入边界 | 极复杂几何快速生成 |

## 网格质量指标

| 指标 | 理想值 | 问题阈值 |
|------|--------|----------|
| 正交性 (Orthogonal Quality) | 1 (完美) | < 0.1 严重问题 |
| 偏斜度 (Skewness) | 0 (完美) | > 0.95 严重问题 |
| 长宽比 (Aspect Ratio) | ~1 | > 100 平板边界层可接受 |
| 增长率 (Growth Rate) | < 1.2 | > 1.5 可能影响精度 |

## 边界层网格

**第一层高度估算**：

$$y_1 = \frac{y^+ \nu}{u_*} \approx \frac{y^+ L}{0.036 Re^{4/5}}$$

| 目标 $y^+$ | 壁面处理策略 |
|-------------|-------------|
| $< 1$ | 求解粘性子层（DNS/LES） |
| $\approx 1$ | $k\text{-}\omega$ SST 解析到粘性子层 |
| $30-300$ | 壁面函数（$k\text{-}\varepsilon$） |
| $< 5$ 或 $>30$ | 避免在缓冲区！ |

## 网格无关性验证

**标准做法**：依次细化网格（每维增大 1.5×），当关键量（升力 $C_l$、压降 $\Delta p$、壁面换热率）变化 < 1% 时，网格分辨率足够。

## 自适应网格 (AMR)

运行时根据流场特征（梯度、曲率、涡量）自动加密/粗化网格：
- 激波附近加密（捕捉间断）
- 尾迹区加密（解析涡结构）
- 远场均匀区粗化（节省计算）
