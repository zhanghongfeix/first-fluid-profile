# 🌊 流体力学知识库

> 系统掌握流体力学基础理论、研究前沿与数值模拟工具

欢迎！这是一个面向流体力学学习者和研究者的开源知识库。

## 📖 内容板块

| 板块 | 说明 |
|------|------|
| 🧮 基础理论 | NS方程、边界层、涡动力学等核心理论 |
| 🔬 研究前沿 | 10个研究方向的最新进展 |
| 💻 CFD教程 | 开源与商业CFD工具上手指南 |
| 📰 最新论文 | 每日自动抓取 arXiv 最新流体力学论文 |
| 📚 术语表 | 流体力学专业术语中英对照 |

## 🚀 快速开始

### 本地运行

```bash
npm install -g docsify-cli
cd 第一个项目
docsify serve .
```

浏览器打开 http://localhost:3000 即可。

### 论文抓取

```bash
pip install arxiv
python scripts/fetch_papers.py
```
